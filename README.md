# scripa
ScrIPA, a new input method for IPA. 

ScrIPA, 一款新设计的IPA输入法。包括了一种独特的、更科学更好记的默认编码方式。

### 使用方法（.exe）

输入字母，会自动转换为IPA，包含了几乎所有常用的IPA国际音标。

按enter键（默认第一个）、大键盘数字键或鼠标点选可以复制到剪贴板。

按空格键可以暂存已有的输出（默认第一个，不想要第一个的可以分开来打字）并且空一格，后面的输入将不影响前面已有的输出。便于打包一起复制粘贴。

### 功能（.exe）
1.可以选择特定的转写编码库是否生效，这可以为特定某些语言的使用者提供便利。

2.可以按大键盘数字键来选字，小键盘数字键来输入需要的数字。（没有小键盘的用户可以选择关掉或重新调节键位）

3.windows 0.0.300beta 已经可以发布。在build里直接运行exe就行。

4.左边书状按钮可以打开默认编码库速记表。【尚未实装】

5.可以通过窗口编辑和客制化自己的custom库。【尚未实装】


### 编码库
default:默认。也就是新转写一字一码的方式。【默认可用】

simple:简记法。是音标的可能常用，惯用的形式。【默认可用】

tones:音调。所有调值符号都在这里。【默认可用】

chinese:汉语模式，使调值、儿化音、送气音等更容易打出。旨在造福汉语语言学学生。

custom:任君选择。
