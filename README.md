# scripa
ScrIPA, a new input method for IPA. 
ScrIPA, 一款新设计的IPA输入法。包括了一种独特的、更科学更好记的默认编码方式。

### 步骤
1.实现可选转写列表功能使其客制化。
2.设计和制作ui。
3.实现可选字和快捷键系统。
4.做一个windows*目前只考虑windows框架,把这个体系装进去。利用TSF框架。
亦预计上线Linux版。
5.加装可视的选词框功能。
6.设计界面，打包，测试。
7.发布

### 编码库
default:默认。也就是新转写一字一码的方式。【默认可用】
simple:音读，简记法。是按照音标的可能常用，惯用的形式。【默认可用】
tones:音调。所有调值符号都在这里。【默认可用】
custom:任君选择。
