# scripa

## 简体中文 / Sim.Chinese

ScrIPA, 一款新设计的IPA输入法。包括了一种独特的、更科学更好记的默认编码方式。

##### v0.0.5已经发布，直接下载，点击scripa_app/build/0.0.5xx.exe运行。只需要下载 scripa_app/一个文件夹的内容即可使用。

### 使用方法

输入字母，会自动转换为IPA，包含了几乎所有常用的IPA国际音标。

按enter键（默认第一个）、大键盘数字键或鼠标点选可以复制到剪贴板。

按空格键可以暂存已有的输出（默认第一个，不想要第一个的可以分开来打字）并且空一格，后面的输入将不影响前面已有的输出。便于打包一起复制粘贴。

### 功能

1.可以按大键盘数字键来选字，小键盘数字键来输入需要的数字。（没有小键盘的用户可以选择关掉或重新调节键位【尚未实装】）

2.左边书状按钮可以打开默认编码库速记表。速记表暂时是手写的，且某些不在表上的音标没有显示，今后会设计更完整、可交互的图表。

3.左二可以展开作者信息。

4.右侧箭头按钮可以翻页。

5.在设置按钮下，可以选择特定的转写编码库是否生效，这可以为某些特定语言的使用者提供便利。

6.在设置按钮下，可以通过窗口编辑和客制化自己的custom库。

### 和弦模式 v0.0.5.2新增

点按钢琴键盘，在备选栏会输出当前和弦。enter键可以复制到剪贴板。

输入音名，每个音名之间请按空格确认。abcdefg。sharp=s, flat=t。确认后备选栏会输出当前和弦，enter键可以复制到剪贴板。

### 编码库
default-default:默认。也就是新转写方式。

default-simple:简记法。是音标的可能常用，惯用的形式。

default-tones:音调。所有调值符号都在这里。

specitalized-chinese:汉语模式，使调值、儿化音、送气音等更容易打出。旨在造福汉语语言学学生。

specialized-music-chord:输入音符的音名，输出和弦。【尚未实装】

custom:任君选择。

## English 

ScrIPA, a new input method for IPA. It features a unique default encoding system that is more scientific and easier to memorize.

##### v0.0.5 has been released. Download the full package directly and run the scripa_app/build/0.0.5xx.exe file.

### How to Use

Type letters and they will automatically be converted into IPA symbols, covering almost all commonly used International Phonetic Alphabet characters.

Press Enter (default: the first candidate), a number key on the main keyboard, or click with the mouse to copy the selected output to the clipboard.

Press the Space bar to temporarily store the current output (default: the first candidate; if you do not want the first one, type separately), insert a space, and continue typing. Subsequent input will not affect the previously stored output. This makes it convenient to batch-copy multiple results at once.

### Features

You can use number keys on the main keyboard to select candidates, and number keys on the numeric keypad to input numbers.
(Users without a numeric keypad can choose to disable or remap these keys 【not yet implemented】.)

The book-shaped button on the left opens the quick-reference table for the default encoding library.
The table is currently handwritten, and some IPA symbols not included in the table are missing. A more complete and interactive chart will be designed in the future.

You can choose whether specific transcription encoding libraries are enabled. This is intended to provide convenience for users of particular languages.

You can edit and customize your own custom library through the application window.

### Encoding Libraries

default-default: Default library, i.e. the new transcription system.

default-simple: Simplified notation, using commonly used and conventional IPA forms.

default-tones: Tones. All tone and pitch contour symbols are included here.

specialized-chinese: Chinese mode. Makes tone values, erhua, aspiration, and related features easier to input. Designed to benefit students of Chinese linguistics.

specialized-music-chord: Input note names and output chords. 【Not yet implemented】

custom: Completely up to you.
