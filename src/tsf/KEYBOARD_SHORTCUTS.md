# Scripa TSF 输入法快捷键

## 📝 输入与选择

| 按键 | 功能 | 说明 |
|------|------|------|
| **A-Z** | 输入字母 | IPA模式下转换为音标，ENG模式直接输入 |
| **Backspace** | 删除最后一个字符 | 清除输入缓冲区的最后一个字符 |
| **Space** | 选择第一个候选 | 快速选择序号1的候选项 |
| **1-9** (大键盘) | 选择候选项 | 选择当前页对应序号的候选 |
| **0-9** (小键盘) | 输入数字 | 作为普通字符输入 |

## 📖 翻页操作

| 按键 | 功能 | 说明 |
|------|------|------|
| **]** | 下一页 | 翻到候选列表的下一页 |
| **[** | 上一页 | 翻到候选列表的上一页 |
| **PageDown** | 下一页 | 与 ] 功能相同 |
| **PageUp** | 上一页 | 与 [ 功能相同 |

## 🔄 模式切换

| 按键 | 功能 | 说明 |
|------|------|------|
| **Ctrl+Shift** | 切换IPA/ENG模式 | 工具栏图标会更新显示当前模式 |

## 🖱️ 鼠标操作

| 操作 | 功能 | 说明 |
|------|------|------|
| **点击候选项** | 选择该候选 | 直接点击候选框选择 |
| **点击模式图标** | 切换模式 | 点击工具栏左侧图标切换IPA/ENG |
| **点击设置图标** | 打开设置菜单 | 点击工具栏右侧设置按钮 (TODO) |
| **点击默认图标** | 打开方案参考 | 点击工具栏中间按钮 (TODO) |

## 💡 使用技巧

### 快速输入流程
1. 输入拉丁字母（如：`ama`）
2. 候选窗口自动弹出
3. 使用方式任选：
   - 按 **Space** 选择第一个
   - 按 **1-9** 选择对应序号
   - 用 **[** 或 **]** 翻页查看更多
   - 鼠标点击候选项

### 跨页选择
```
输入: "a" → 候选项很多
操作: 按 ] 翻页 → 找到目标 → 按数字键选择
```

### 数字输入
- **大键盘数字**: 仅在有候选窗口时用于选择，否则正常输入
- **小键盘数字**: 始终作为字符输入，可能触发IPA转换

## 🎯 候选窗口说明

### 显示内容
```
┌─────────────────────────────────┐
│ Buffer: ama                     │  ← 当前输入缓冲
├─────────────────────────────────┤
│ 1 ā  2 á  3 ǎ  4 à  5 a  ...   │  ← 候选项（序号1-9）
├─────────────────────────────────┤
│         Page 2/5                │  ← 页码指示
└─────────────────────────────────┘
```

### 行为特性
- 自动跟随光标位置
- 每页最多显示8个候选项
- 蓝色高亮表示选中项（鼠标悬停）
- 无候选项时自动隐藏

## 🛠️ 工具栏说明

工具栏位于屏幕右上角：
```
┌──────────────────┐
│ [IPA] [DEF] [⚙] │
└──────────────────┘
   ▲      ▲     ▲
   │      │     └─ 设置按钮
   │      └─────── 方案参考
   └──────────── 模式指示器（点击切换）
```

## 🔧 调试提示

- **候选窗口未显示**: 检查输入缓冲是否有内容
- **翻页无反应**: 确认候选列表超过8个
- **数字键不选择**: 可能使用了小键盘，应使用大键盘
- **Ctrl+Shift无效**: 重新注册TSF输入法并重启explorer
